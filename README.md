# MetroloJA

A guide for time-tracking metrology analyses of light microscopes, after using the ImageJ MetroloJ_QC plugin.

MetroloJ_QC is a plugin whose goal is to enable automation of quality control tests regularly implemented within a photonic microscopy platform.
